# 웨어러블 기반 보험료 할인 모델 설계 및 시뮬레이션 분석 (초코하임)

본 프로젝트는 2025년 DB 보험금융공모전 참가 목적으로 진행된 프로젝트이며, 보험/은행 분야에 지원하였습니다.

웨어러블 데이터(걸음 수, 수면 시간, 수면 패턴)를 활용해 가입자 노력 점수를 산출하고, 이를 보험료 할인 구조에 반영했을 때 보험사 손익이 어떻게 변하는지 분석한 프로젝트입니다.

## 1. 문서 안내
- 논문 내용은 `초코하임.pdf`에서 확인할 수 있습니다.

## 2. 연구 개요
- 목표: 웨어러블 기반 노력 점수를 보험료 할인 함수에 결합해 손익/안정성 변화를 평가
- 데이터: NHANES 기반 생체지표 + 활동/수면 데이터
- 핵심 방법:
  - 15개 생체지표 기반 K-Means 위험군 분류(3개 군)
  - 3개 도메인(걸음 수, 수면 시간, 수면 패턴) 노력 점수 산출
  - 할인 상한(5/10/15/20%) 시나리오별 손익 비교
  - 부트스트랩(5,000회) + t-test로 결과 안정성 검증

## 3. 핵심 결과 요약
할인 상한이 커질수록 평균 보험료와 평균 기대이익은 감소하지만, 손익 분포의 변동성은 완화되는 경향을 확인했습니다.

| 할인 상한 | 평균 보험료 | 평균 기대손해액 | 평균 할인액 | 평균 기대이익 |
| --- | ---: | ---: | ---: | ---: |
| 5%  | 60,862 | 6,670 | 1,355 | 54,192 |
| 10% | 59,508 | 6,670 | 2,709 | 52,838 |
| 15% | 58,153 | 6,670 | 4,064 | 51,483 |
| 20% | 56,798 | 6,678 | 5,418 | 50,129 |

추가로, 전체 손익은 고위험군 할인율 자체보다 저위험군/중위험군의 보험료와 가입자 수에 더 큰 영향을 받는 패턴을 보였습니다.
