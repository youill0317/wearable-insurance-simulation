# 웨어러블 기반 보험료 할인 모델 설계 및 시뮬레이션 분석 (초코하임)

본 프로젝트는 2025년 DB 보험금융공모전 참가 목적으로 진행된 프로젝트이며, 보험/은행 분야에 지원하였습니다.

웨어러블 데이터(걸음 수, 수면 시간, 수면 패턴)를 활용해 가입자 노력 점수를 산출하고, 이를 보험료 할인 구조에 반영했을 때 보험사 손익이 어떻게 변하는지 분석한 프로젝트입니다.

## 1. 문서 안내

- 논문 내용은 `초코하임.pdf`에서 확인할 수 있습니다.
- 추가 스윗스팟 연구의 코드와 실행 결과는 `wearable_insurance_sweet_spot_study.ipynb`에서 확인할 수 있습니다.

## 2. 연구 개요

- 목표: 웨어러블 기반 노력 점수를 보험료 할인 함수에 결합해 손익/안정성 변화를 평가
- 데이터: NHANES 기반 생체지표 + 활동/수면 데이터
- 핵심 방법:
  - 15개 생체지표 기반 K-Means 위험군 분류(3개 군)
  - 3개 도메인(걸음 수, 수면 시간, 수면 패턴) 노력 점수 산출
  - 할인 상한(5/10/15/20%) 시나리오별 손익 비교
  - 부트스트랩(5,000회) + t-test로 결과 안정성 검증

## 3. 공모전 당시 결과 요약

아래 결과는 할인율과 보험금 감소를 인과적으로 연결하기 전의 초기 시뮬레이션입니다.

할인 상한이 커질수록 평균 보험료와 평균 기대이익은 감소하지만, 손익 분포의 변동성은 완화되는 경향을 확인했습니다.

| 할인 상한 | 평균 보험료 | 평균 기대손해액 | 평균 할인액 | 평균 기대이익 |
| --- | ---: | ---: | ---: | ---: |
| 5%  | 60,862 | 6,670 | 1,355 | 54,192 |
| 10% | 59,508 | 6,670 | 2,709 | 52,838 |
| 15% | 58,153 | 6,670 | 4,064 | 51,483 |
| 20% | 56,798 | 6,678 | 5,418 | 50,129 |

추가로, 전체 손익은 고위험군 할인율 자체보다 저위험군/중위험군의 보험료와 가입자 수에 더 큰 영향을 받는 패턴을 보였습니다.

## 4. 추가 연구: 보험료 할인 스윗스팟

### 4.1 기존 접근의 한계

- 5/10/15/20% 할인 비교에서 기대손해가 약 6,670원으로 거의 고정되어, 할인이 행동과 보험금 감소를 유도하는 과정이 없었습니다.
- 약 7천 명의 자료를 1,500만 개 IID 합성행으로 늘려 개인 내 상관, 요일효과와 변수 간 공분산을 잃었습니다.
- 합성 bootstrap 반복을 독립 표본처럼 검정해 실제 표본불확실성보다 Monte Carlo 오차를 정밀하게 추정했습니다.
- K-Means 위험군은 silhouette 0.1479, 질병력 Cramér's V 0.091–0.113으로 분리가 약했고 미래 보험금 모형이 아니었습니다.
- 서로 다른 연구 endpoint에서 만든 `theta`, 고정 `eta=1`, 운영비 0원 등 결과를 크게 좌우하는 가정이 최적화와 분리되지 않았습니다.

추가 연구에서는 이를 다음 흐름으로 교체했습니다.

```text
할인 → 참여·증분 걸음 → CVD 위험 변화 → 조정 가능한 보험금 감소
```

### 4.2 데이터와 분석 설계

- 기존 NHANES 2011–2014 G/H 자료와 PhysioNet 분 단위 step 파생자료만 사용했습니다.
- 대용량 1,440분 자료는 Modal의 8 CPU·16GB 메모리에서 원격 집계했습니다.
- 불완전한 측정 day 1·9를 제외하고 wake-wear 600분 이상, 최소 4개 유효일을 요구했습니다.
- 집계된 성인 8,023명 중 일평균 500–30,000보 범위의 7,934명을 최종 분석했습니다.
- G cycle에서 위험모형과 정책을 선택하고 H cycle은 손대지 않은 최종 검증 표본으로 유지했습니다.
- 위험모형은 logistic regression과 HistGradientBoosting, 임상정보 단독과 임상+웨어러블 조합을 비교했습니다.
- 할인상한 0–5%, 개인 목표 +500/+1,000/+1,500보, 위험대상 전체/상위 50%/상위 25%를 grid 탐색했습니다.
- 표본불확실성은 NHANES stratum 내 PSU bootstrap, 구조불확실성은 Monte Carlo와 `m–κ` 민감도로 분리했습니다.

경제 목적함수는 다음과 같습니다.

```text
보험사 순편익 = 기대 보험금 절감 - 실제 할인 - 기기·운영비
```

여기서 `m`은 `portfolio loss ratio × 걸음으로 바뀔 수 있는 청구 비중`, `κ`는 관찰연구의 건강효과를 실제 인과효과로 전이할 때의 축소계수입니다. 기본값은 기존 연구와 연결되는 `m=12.7048%`, 보수적 `κ=0.25`이며, 운영비 0원은 할인에 가장 유리한 상한 분석입니다.

### 4.3 핵심 결과

기본 시나리오의 연구 스윗스팟은 다음과 같습니다.

| 항목 | 결과 |
| --- | ---: |
| 할인 상한 | **0.15%** |
| 대상 | baseline 7,000보 미만 전체 위험군 |
| 개인 목표 | baseline 대비 +1,500보/일 |
| H 평균 보험금 절감 | 0.2017원/포트폴리오 인월 |
| H 평균 할인 | 0.1106원/포트폴리오 인월 |
| H 평균 순편익 | **+0.0911원/포트폴리오 인월** |
| 가입자당 실제 평균 할인 | 0.66원/월 |
| 가입자당 평균 행동증가 | 9.3보/일 |
| 감당 가능한 최대 운영비 | 0.54원/가입자월 |

따라서 **0.15%는 현재 데이터와 문헌을 결합한 기본 시나리오의 기술적 스윗스팟이자 후속 pilot 기준값**입니다. 0%보다 순편익이 높았지만 효과와 보상이 매우 작아 상용 상품 권고값으로 바로 해석할 수는 없습니다.

체감 가능한 보상에는 손익이 뒤집혔습니다.

| 정책 | H 평균 순편익 |
| --- | ---: |
| 2.5% 상한, +500보, 고위험 25% | -139.98원/포트폴리오 인월 |
| 5% 상한, +1,000보, 고위험 25% | -178.67원/포트폴리오 인월 |

추가 결과는 다음과 같습니다.

- 임상 logistic의 H AUROC는 0.8323이었고 웨어러블을 추가하면 0.8343으로 +0.0020만 증가했습니다. 현재 상태보다 **검증된 증분 개선**을 할인 근거로 삼는 편이 타당합니다.
- 고정 가정의 H PSU bootstrap 95% 구간은 +0.0880~+0.0945원이었지만, 구조 Monte Carlo의 `P(net>0)`는 운영비 0원에서도 70.9%였습니다.
- 가입자당 운영비를 1원만 반영하면 `P(net>0)`는 27.7%, 100원이면 0%였습니다.
- 체감 정책이 `κ=0.25`에서 손익분기하려면 `m=286.5%`가 필요했고, 기존 `m=12.7%`에서는 `κ=1`이어도 손익분기하지 못했습니다.
- 7–9시간 수면 상태에 추가 bonus를 주면 건강절감 근거 없이 20.19원/포트폴리오 인월의 deadweight 비용이 더해졌습니다.

### 4.4 해석

- NHANES 같은 횡단면 자료를 더 추가하면 표본오차는 줄지만 0.15%가 크게 변할 가능성은 상대적으로 낮습니다.
- 실제 paid claims, earned premium, 운영비 또는 무작위 할인 pilot 자료가 추가되면 `m`, `κ`, 할인→행동 탄력성이 바뀌므로 최적 상한도 달라질 수 있습니다.
- 민감도 분석에서 최적 상한은 0–1.45%까지 이동했습니다. 따라서 0.15%는 보편 상수가 아니라 현재 기본 시나리오의 결과입니다.
- 상품 출시 전에는 저활동·고위험층을 대상으로 0/0.5/1% 개선보상을 무작위 비교하고, baseline 4주·12개월 행동·후속 paid claims·탈락·기기오차·운영비를 함께 기록해야 합니다.
- 의료적 활동제약에는 동등한 대체목표를 제공하고 비참여자를 할증하지 않아야 합니다.
